#  Sitio Web de Turismo

Este proyecto consiste en el desarrollo de un sitio web de turismo utilizando HTML5 y CSS3.

##  Descripción

El sitio web presenta diferentes secciones relacionadas con el turismo en Argentina, incluyendo destinos, agencias de viajes, precios y más. 

## Estructura del proyecto

- home.html → Página principal
- destinos.html → Destinos turísticos
- agencias.html → Agencias de viaje 
- precios.html → Comparación de precios
- blog.html → Blog de destinos
- css → Estilos del sitio
